.. _installer.scripts:

=================
Script Generation
=================

``installer.scripts`` provides the ability to generate executable launcher
scripts, that are based on `simple_launcher`_'. A description of how these
scripts work is available in simple_launcher's README.

.. _`simple_launcher`: https://bitbucket.org/vinay.sajip/simple_launcher/
